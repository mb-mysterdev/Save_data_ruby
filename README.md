# Save data
Notre programme permet de sauvegarder les listes scrappées sur le site de la mairie du Val d'Oise

# Instructions 
Pour tester le programme, nous vous conseillons d'utiliser Binding.pry. Créez une instance de classe (exemple : toto=TownHallScrapper.new), puis appliquez à cette instance la méthode que vous voulez tester (exemple: toto.save_as_spreadsheet).  

Nous n'avons pas eu le temps de créer app.rb pour appeler les différentes méthodes. Afin de remplir la spreadsheet avec save_as_spreadsheet, créez votre propre spreadsheet d'abord puis entrez la clé d'url de votre spreadsheet dans la méthode.
